# System.String Extensions

Below are examples of the .NET ``System.String`` extension methods in TenjinX.

[Home](../../../README.md)

```c#
using TenjinX.Extensions;

// GetFullName
typeof(string).GetFullName();   // "System.String"
typeof(int).GetFullName();      // "System.Int32"
typeof(DateTime).GetFullName(); // "System.DateTime"
```
